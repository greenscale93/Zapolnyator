# Планы на развитие

## 🔴 Критические (1 приоритет)

### 1. Добавить расчёт прибыли без учёта отклонений по ФОТ
Для быстрой сверки и контроля нужны показатели:
- Прибыль по актам без учёта отклонений ФОТ относительно прошлого месяца
- Прибыль по ДС без учёта отклонений ФОТ относительно прошлого месяца

## 🟡 Важные (2 приоритет)

### 2. Постоянное хранение маппинга
Сейчас маппинг (соответствие контрагент → офис) хранится в SQLite внутри контейнера.
При перезапуске контейнера данные теряются, и приходится заново заполнять маппинг в диалоге.
Нужно вынести БД на постоянный volume или внешнее хранилище.

### 4. Применять фильтры на листах
Добавить возможность применения автофильтров к заполненным листам (через openpyxl или LO)
для удобства просмотра результата.

## 🟢 Низкий приоритет

### 5. Устанавливать пароль обратно
После обработки файл теряет пароль (расшифровывается для обработки, но не шифруется обратно).
Нужно устанавливать пароль обратно на выходной файл.
